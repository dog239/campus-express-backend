# 后端接口修改建议 - 订单隐私保护

## 修改目的
保护用户隐私，未接单的订单不返回取件码信息。

---

## 2. 接单接口 `/api/order/accept/{orderId}`

### 修改前
```java
@PostMapping("/accept/{orderId}")
public Result<Map<String, Object>> accept(
    @RequestHeader("Authorization") String authorization,
    @PathVariable Long orderId) {
    
    Order order = orderService.accept(orderId, getCurrentUserId(authorization));
    Map<String, Object> result = new HashMap<>();
    result.put("orderId", order.getId());
    result.put("status", order.getStatus());
    return Result.success(result);
}
```

### 修改后
**OrderController.java**
```java
@PostMapping("/accept/{orderId}")
public Result<Map<String, Object>> accept(
    @RequestHeader("Authorization") String authorization,
    @PathVariable Long orderId) {
    
    Long userId = getCurrentUserId(authorization);
    Order order = orderService.accept(orderId, userId);
    
    // 接单成功后，返回取件码
    List<String> pickupCodes = orderService.getPickupCodesByOrderId(orderId);
    String stationName = orderService.getStationNameByOrderId(orderId);
    
    Map<String, Object> result = new HashMap<>();
    result.put("orderId", order.getId());
    result.put("status", order.getStatus());
    result.put("pickupCodes", pickupCodes);  // 返回取件码
    result.put("stationName", stationName);  // 返回驿站名称
    
    return Result.success(result);
}
```

**OrderService.java** - 新增方法
```java
public List<String> getPickupCodesByOrderId(Long orderId) {
    // 查询订单关联的包裹，获取所有取件码
    List<ExpressPackage> packages = packageMapper.selectList(
        new QueryWrapper<ExpressPackage>()
            .in("id", getPackageIdsByOrderId(orderId))
    );
    
    return packages.stream()
        .map(ExpressPackage::getPickupCode)
        .collect(Collectors.toList());
}

public String getStationNameByOrderId(Long orderId) {
    // 查询订单关联的包裹，获取驿站名称
    ExpressPackage pkg = packageMapper.selectOne(
        new QueryWrapper<ExpressPackage>()
            .in("id", getPackageIdsByOrderId(orderId))
            .last("LIMIT 1")
    );
    
    return pkg != null ? pkg.getStationName() : null;
}
```

## 3. 订单详情接口 `/api/order/detail/{orderId}`

### 修改前
```java
@GetMapping("/detail/{orderId}")

    return Result.success(orderDetail);

```

### 修改后
**OrderController.java**
```java
@GetMapping("/detail/{orderId}")
public Result<OrderDetailVO> getDetail(
    @RequestHeader(value = "Authorization", required = false) String authorization,
    @PathVariable Long orderId) {
    
    Long currentUserId = getCurrentUserId(authorization);
    OrderDetailVO orderDetail = orderService.getOrderDetail(orderId);
    
    // 如果订单未接单，且当前用户不是发布者，隐藏取件码
    if (orderDetail.getStatus() == 0 && 
        !orderDetail.getRequesterId().equals(currentUserId)) {
        orderDetail.setPickupCodes(null);  // 隐藏取件码
        orderDetail.setPackageIds(null);   // 隐藏包裹ID
    }
    
    return Result.success(orderDetail);
}
```

## 5. 数据库查询优化

### OrderService.java
```java
public List<AvailableOrderVO> getAvailableOrders() {
    // 查询待接单订单，不关联包裹表
    List<Order> orders = orderMapper.selectList(
        new QueryWrapper<Order>()
            .eq("status", 0)
            .orderByDesc("create_time")
    );
    
    return orders.stream()
        .map(order -> {
            AvailableOrderVO vo = new AvailableOrderVO();
            vo.setOrderId(order.getId());
            vo.setTipAmount(order.getTipAmount());
            vo.setCreateTime(order.getCreateTime());
            
            // 获取发布者信息
            User requester = userMapper.selectById(order.getRequesterId());
            vo.setRequesterNickname(requester.getNickname());
            
            // 获取驿站名称和包裹数量（不返回取件码）
            vo.setStationName(getStationName(order.getPackageIds()));
            vo.setPackageCount(getPackageCount(order.getPackageIds()));
            
            return vo;
        })
        .collect(Collectors.toList());
}

private String getStationName(String packageIds) {
    if (packageIds == null || packageIds.isEmpty()) {
        return null;
    }
    
    Long firstPackageId = Long.parseLong(packageIds.split(",")[0]);
    ExpressPackage pkg = packageMapper.selectById(firstPackageId);
    return pkg != null ? pkg.getStationName() : null;
}

private Integer getPackageCount(String packageIds) {
    if (packageIds == null || packageIds.isEmpty()) {
        return 0;
    }
    return packageIds.split(",").length;
}
```

## 6. 安全验证

### OrderController.java
```java
private Long getCurrentUserId(String authorization) {
    if (authorization == null || authorization.isEmpty()) {
        throw new IllegalArgumentException("未授权");
    }
    
    String openid = jwtUtil.parseToken(authorization);
    User user = userService.getUserByOpenid(openid);
    
    if (user == null) {
        throw new IllegalArgumentException("用户不存在");
    }
    
    return user.getId();
}
```

## 8. 注意事项

1. **未接单订单**：绝对不能返回 `pickupCode` 字段
2. **接单成功**：立即返回取件码，方便用户取件
3. **我的订单**：可以显示取件码（已接单）
4. **订单详情**：根据用户身份判断是否显示取件码
5. **日志记录**：记录取件码查看行为，便于审计
