### 2. UserController.java - bindPhone 接口
```java
@PostMapping("/bindPhone")
public Result<Map<String, String>> bindPhone(
        @RequestHeader("Authorization") String authorization,
        @RequestBody Map<String, String> request) {
    try {
        String encryptedData = request.get("encryptedData");
        String iv = request.get("iv");
        String code = request.get("code");
        String phone = request.get("phone");
        
        String openid = jwtUtil.parseToken(authorization);
        
        String boundPhone;
        
        if (phone != null && !phone.trim().isEmpty()) {
            // 直接绑定手机号（测试用）
            boundPhone = userService.bindPhoneDirect(openid, phone);
        } else if (encryptedData != null && iv != null && code != null) {
            // 微信加密数据解密
            boundPhone = userService.bindPhone(openid, encryptedData, iv, code);
        } else {
            return Result.error("请提供手机号或微信加密数据");
        }
        
        Map<String, String> result = new HashMap<>();
        result.put("phone", boundPhone);
        return Result.success(result);
    } catch (Exception e) {
        return Result.error(e.getMessage());
    }
}
```

### 3. UserService.java - 核心方法

#### bindPhone 方法（微信加密数据解密）
```java
public String bindPhone(String openid, String encryptedData, String iv, String code) {
    // 1. 获取 session_key
    String sessionKey = getSessionKey(code);
    
    // 2. 解密手机号
    String phone = decryptPhoneNumber(encryptedData, sessionKey, iv);
    
    // 3. 保存到数据库
    userMapper.update(null, new UpdateWrapper<User>()
            .eq("openid", openid)
            .set("phone", phone)
            .set("update_time", LocalDateTime.now()));
    
    return phone;
}
```

#### getSessionKey 方法
```java
private String getSessionKey(String code) {
    String url = String.format("%s?appid=%s&secret=%s&js_code=%s&grant_type=authorization_code",
            wechatProperties.getSessionUrl(),
            wechatProperties.getAppid(),
            wechatProperties.getSecret(),
            code);
    
    String response = restTemplate.getForObject(url, String.class);
    JSONObject json = JSONObject.parseObject(response);
    
    if (json.containsKey("session_key")) {
        return json.getString("session_key");
    } else {
        throw new RuntimeException("获取 session_key 失败");
    }
}
```

#### decryptPhoneNumber 方法（AES 解密）
```java
private String decryptPhoneNumber(String encryptedData, String sessionKey, String iv) {
    // Base64 解码
    byte[] sessionKeyBytes = Base64.getDecoder().decode(sessionKey);
    byte[] ivBytes = Base64.getDecoder().decode(iv);
    byte[] encryptedBytes = Base64.getDecoder().decode(encryptedData);
    
    // AES/CBC/PKCS5Padding 解密
    SecretKeySpec keySpec = new SecretKeySpec(sessionKeyBytes, "AES");
    IvParameterSpec ivSpec = new IvParameterSpec(ivBytes);
    
    Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");
    cipher.init(Cipher.DECRYPT_MODE, keySpec, ivSpec);
    
    byte[] decrypted = cipher.doFinal(encryptedBytes);
    String decryptedStr = new String(decrypted, StandardCharsets.UTF_8);
    
    // 解析 JSON 提取手机号
    JSONObject json = JSONObject.parseObject(decryptedStr);
    String phone = json.getString("phoneNumber");
    
    return phone;
}
```

### pages/user/index.js
```javascript
async handleGetPhoneNumber(e) {
  if (e.detail.errMsg === 'getPhoneNumber:ok') {
    wx.showLoading({ title: '绑定中...', mask: true })
    
    try {
      const { encryptedData, iv, code } = e.detail
      
      // 重新获取 code（重要！）
      const loginRes = await new Promise((resolve, reject) => {
        wx.login({ success: resolve, fail: reject })
      })
      
      // 调用后端接口
      const result = await post('/api/user/bindPhone', {
        encryptedData,
        iv,
        code: loginRes.code
      })
      
      // 更新本地数据
      const userInfo = this.data.userInfo
      userInfo.phone = result.phone
      
      this.setData({ userInfo, hasPhone: true })
      wx.setStorageSync('userInfo', userInfo)
      
      wx.showToast({ title: '绑定成功', icon: 'success' })
    } catch (err) {
      wx.showToast({ title: err.message || '绑定失败', icon: 'none' })
    } finally {
      wx.hideLoading()
    }
  }
}
```

## 四、常见问题

### Q1: 解密失败 - "解密手机号失败: javax.crypto.BadPaddingException"
**原因**：session_key 过期或 code 不匹配
**解决**：确保每次调用时都重新执行 `wx.login()` 获取新的 code

### Q2: 微信接口返回 errcode: 40029
**原因**：code 无效或已使用
**解决**：code 只能使用一次，必须重新获取

### Q3: 小程序报错 "getPhoneNumber:fail no permission"
**原因**：小程序未开通手机号权限
**解决**：
1. 登录微信公众平台
2. 进入"开发管理" -> "接口设置"
3. 开通"获取用户手机号"权限

### Q4: 后端报错 "缺少 token"
**原因**：用户未登录或 token 过期
**解决**：确保用户已登录，检查 Authorization header

## 五、注意事项

1. **code 必须重新获取**：`getPhoneNumber` 返回的 code 不能用于解密，必须重新调用 `wx.login()`
2. **session_key 有效期**：session_key 有时效性，建议实时获取
3. **真机测试**：开发者工具可能无法获取真实手机号，建议真机测试
4. **隐私协议**：小程序需要配置隐私保护指引
5. **数据库字段**：确保 user 表有 phone 字段（varchar(20)）
